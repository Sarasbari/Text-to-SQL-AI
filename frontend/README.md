# Text-to-SQL Frontend

This folder contains the React, Vite, TypeScript, Tailwind, and Monaco Editor frontend for the query workbench.

## Features

- **Schema Browser:** Visual treeview of tables, columns, and foreign key references.
- **SQL Workbench:** Interactive Monaco Editor configured for SQL syntax.
- **Data Results Panel:** Execution statistics, paginated table, and CSV exporter.
- **Recovery Loop Stepper:** Visual timeline displaying the auto-correction stages.

## How to Run

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start development server:
   ```bash
   npm run dev
   ```
